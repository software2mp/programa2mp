Contexto Documentos

El Contexto Documentos contiene textos que desarrollan teóricamente conceptos afines a la temática del Producto Autocontenido Temático. También es posible copiar elementos de un contexto a otro, siempre que soporten el formato correspondiente.



Elementos del Contexto Documentos

Documento. Elementos con textos para trabajar con el PAT.



Menú

Nuevo PAT. Limpia todos los contextos cerrando los elementos cargados.

Administración de PATs. Despliega la ventana de administración de PATs

Guardar PAT. Guarda los elementos cargados en todos los contextos como PAT de usuario.

Agregar Elemento. Permite seleccionar un elemento de texto en formato HTML.

Cerrar Elemento. Se activa al seleccionar un elemento de la lista y lo quita.

Agrupar los elementos seleccionados. Genera en la lista de materiales un grupo donde incorporar coberturas vectoriales temáticas.
